# 它是如何工作的？
学习该项目不仅可以，使用它轻松的创建美观的GUI界面，也可以更深层次的学习计算机GUI的底层原理。

## 基层
`glfw`是一个用于创建窗口和处理输入的库，并且得益于它的轻量、跨平台，我们就选择了它来创建窗口（后续或许会做对`SDL2`的支持）。
`opengl`为我们提供了后端支持（至于为什么不用`Vulkan`，是因为做关于`Python`绑定的开发者摆烂了）。
`skia`是一个`2D`绘图库，能够较为轻松高速地绘制各种界面。并且它也提供了对`glfw`、`opengl`的支持。

三者结合变形成了如今的`suzaku`。
> `glfw`创建窗口，`opengl`提供后端支持，`skia`提供绘图接口。

正因为它是用`skia`绘制的，所以各种功能都需要仿系统组件制作，没有系统组件的支持，难度大大增加。
如布局、事件处理等，一个小小的文本输入框都要开发几天。

深入学习当然需要对这几个框架有一定的了解。从而帮助你自己开发各种功能、绘制各种组件。
