class suzaku.event.SkEventHandling[source]

A class containing event handling abilities.

This class should be inherited by other classes with such abilities.

Events should be represented in the form of event_type or event_type[args]. e.g. delay or delay[500]

EVENT_TYPES: list[str] = ['resize', 'move', 'configure', 'update', 'redraw', 'mouse_move', 'mouse_enter', 'mouse_leave', 'mouse_press', 'mouse_release', 'click', 'focus_gain', 'focus_loss', 'key_press', 'key_release', 'key_repeat', 'char', 'delay', 'repeat']

bind(event_type: str, target: Callable | Iterable, multithread: bool = False, _keep_at_clear: bool = False) → SkBoundTask | bool[source]

To bind a task to the object when a specific type of event is triggered.

Example

This shows binding a hello world to the button when it’s press.

Parameters:

  • event_type – The type of event to be bound to

  • target – A (list of) callable thing, what to do when this task is executed

  • multithread – If this task should be executed in another thread (False by default)

  • _keep_at_clear – If the task should be kept when cleaning the event’s binding

Returns:

SkBoundTask that is bound to the task if success, otherwise False
