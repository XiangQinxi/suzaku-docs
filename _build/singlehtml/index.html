下载与安装¶

第一步是下载Suzaku。您可以通过pip轻松下载：

pip install suzaku

但请注意，Suzaku依赖于pyopengl、skia-python和glfw库。

Linux用户可能需要运行apt install libglfw3来安装glfw库。

运行apt-get install libfontconfig1 libgl1-mesa-glx libgl1-mesa-egl libegl1 libglvnd0 libgl1-mesa-dri来安装skia-python的依赖项。

安装完成后，您可以通过运行python -m suzaku来启动Suzaku。

Download & Install¶

The first step is to download Suzaku. You can easily download it using pip:

pip install suzaku

But note that Suzaku depends on pyopengl, skia-python, and glfw libraries.

Linux users may need to run apt install libglfw3 to install glfw library.

apt-get install libfontconfig1 libgl1-mesa-glx libgl1-mesa-egl libegl1 libglvnd0 libgl1-mesa-dri to install skia-python dependencies.

Example¶

After that, you can run Suzaku by running python -m suzaku.

How it works?¶

这里有汉化版本

它是如何工作的？¶

学习该项目不仅可以，使用它轻松的创建美观的GUI界面，也可以更深层次的学习计算机GUI的底层原理。

基层¶

glfw是一个用于创建窗口和处理输入的库，并且得益于它的轻量、跨平台，我们就选择了它来创建窗口（后续或许会做对SDL2的支持）。 opengl为我们提供了后端支持（至于为什么不用Vulkan，是因为做关于Python绑定的开发者摆烂了）。 skia是一个2D绘图库，能够较为轻松高速地绘制各种界面。并且它也提供了对glfw、opengl的支持。

三者结合变形成了如今的suzaku。

glfw创建窗口，opengl提供后端支持，skia提供绘图接口。

正因为它是用skia绘制的，所以各种功能都需要仿系统组件制作，没有系统组件的支持，难度大大增加。 如布局、事件处理等，一个小小的文本输入框都要开发几天。

深入学习当然需要对这几个框架有一定的了解。从而帮助你自己开发各种功能、绘制各种组件。
